测试用例名称：完成用户登录
测试目的：验证用户能通过输入手机号、勾选协议、点击登录按钮进入工作台

起始 URL：https://tpt.supcon.com/tpt-app/#/home/chat/main?TptSaasUserTenantryId=ATL43NW8

测试步骤：

步骤1: 完成用户登录
- 在「请输入用户名」输入框中输入手机号，确认用户名输入框 value 变为"15700078644"，密码输入框 value 变为带掩码的密码
- 勾选「同意协议」复选框，确认复选框"agree"checked 状态从 false 变为 true
- 点击可见文本为「立即登录」的按钮，确认页面跳转至工作台，可见「我的对话」「新建对话」等主界面元素，原登录表单不再可见

步骤2: 打开偏好设置
- 点击用户头像（img 元素，位于"预测预警"下拉区域内），确认用户头像下拉菜单可见，包含"余泽超"、"黄金会员"、"偏好设置"等菜单项
- 点击下拉菜单中可见文本为"偏好设置"的菜单项，确认"偏好设置"对话框出现，包含外观、模式、字体大小、首页显示内容等设置区域

步骤3: 切换显示模式与刷新推荐问题
- 点击"偏好设置"对话框中的"详细模式"选项，确认"偏好设置"对话框中"详细模式"被标记为"使用中"，且页面出现 loading 加载状态
- 点击偏好设置面板中的「推荐问题」开关，确认偏好设置中的「推荐问题」开关变为勾选状态
- 点击「换一批」按钮，确认推荐问题列表内容更新，输入框占位文本同步更换
- 点击「换一批」按钮的刷新图标，确认推荐问题列表内容再次更新，输入框占位文本同步更换

步骤4: 选择专家模式与具体专家
- 点击文本为「专家模式」的选项，确认主区域显示完整的专家列表，包含「AI 操作优化专家」「TPT操作参数优化」「工业知识专家」等条目
- 点击文本为「TPT操作参数优化」的专家条目，确认页面进入 AI 操作优化专家对话界面，输入框 placeholder 显示为「怎么降低乙烯装置的能耗」，推荐问题列表为乙烯装置优化相关

步骤5: 发送对话消息
- 在 placeholder 为「怎么降低乙烯装置的能耗」的输入框中输入「请帮我整定PID」，确认聊天输入框中显示已输入的文字「请帮我整定PID」
- 在聊天输入框（id 为 chat_input_textarea）中输入文字「请帮我整定PID」，确认聊天输入框显示已输入文字「请帮我整定PID」，聊天区域出现对应的新建会话条目及 AI 回答文本

步骤6: 进入我的Agent列表
- 点击可见文本为「我的Agent」的 div 元素（位于侧边导航区域），确认页面主区域由对话界面切换为 Agent 列表视图，可见多个 Agent 卡片及其「执行」按钮

步骤7: 切换至组态生成模式
- 点击 segmented control 中可见文本为「组态生成」的 radio 选项，确认点击未引起可见界面变化（snapshot 一致），表单状态显示「对话生成」选项被选中、「组态生成」选项被取消选中
- 点击「组态生成」单选项，确认「组态生成」单选项显示为已选中（checked），「对话生成」显示为未选中

步骤8: 筛选控制类MPC Agent
- 点击分类标签中可见文本为「控制」的项，确认列表区域显示类型为「控制」的 Agent 条目（如 control_assess_1、AI_Control_1、mpc_2、mpc_1 等）
- 点击子分类中可见文本为「MPC」的项，确认列表区域显示子类型为「MPC」的 Agent 条目（如 mpc_2、mpc_1 等），AI 类型条目不再显示

步骤9: 打开新增Agent对话框并填写名称
- 点击可见文本为「新增Agent」的元素，确认新增 Agent 对话框出现在页面上，可见表单字段与「取 消」「确 认」按钮
- 在 placeholder 为「输入Agent 名称」的输入框中输入内容，确认Agent 名称输入框中显示已输入的 "new_mpc_agent"

步骤10: 输入对应装置值
- 在「对应装置」下拉搜索输入框中输入内容，确认「对应装置」combobox 中显示已输入的值 "A"

步骤11: 确认 Agent 表单提交
- 点击「确 认」按钮，确认Agent 创建/确认弹窗关闭，页面上出现名为"supgpt-obp-iframe"的 Iframe
